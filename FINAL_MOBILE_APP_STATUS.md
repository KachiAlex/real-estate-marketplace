### Option 3: Local Build (Android Studio)

**Prerequisites**:
1. Install Android Studio from https://developer.android.com/studio
2. Install Android SDK Platform 33+
3. Java JDK 11 or 17 (usually comes with Android Studio)

**Build Steps**:
```powershell
cd C:\real-estate-marketplace\mobile-app
npx expo prebuild -p android --clean
cd android
.\gradlew.bat assembleRelease
```

**Result**: APK file in `android/app/build/outputs/apk/release/`

**Time**: ~1 hour for first-time setup + build
